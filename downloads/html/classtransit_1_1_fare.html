Detailed Description

An immutable Value Object representing the cost, or 'fare,' required to ride a TransitVehicle on a particular Route.

Although BusBuddy does not specify explicit rules governing the regularFare and the discountedFare, each TransitProvider must publicly state its Fare policies on a website or other public forum.

Invariant
if discountedFare exists, discountedFare <= regularFare
