Detailed Description

An iTeam implementation of the TripService that exposes Trip data via a REST Service.

Note: The actual Trip calculation algorithm is not specified here and is beyond the scope of this project.

Errors should be communicated via standard HTTP error codes. The common.BaseController will be responsible for mapping Exception types to HTTP error codes, so actual error codes are not specified explicitly here.
